# Multimedia

A collection of multimedia service applications built with Spring Boot.

## Projects

### [Melody](./Melody/)
A music management application consisting of:
- **melody-service**: REST API backend service for managing artists, songs, users, roles, and contacts
- **melody-client**: Web frontend application that consumes the melody-service API

## Technology Stack

- **Java**: 21
- **Spring Boot**: 3.5.7
- **Build Tool**: Maven
